What is it for

This website is dedicated to Miraculous: Tales of Ladybug & Cat Noir.

If you want to watch something but just can't decide what, go to the main page (Get an episode) and get a recommendation based on your preference.

If you want to make sure that your favorite character is getting enough lines click Statistics, choose desired category and learn a bit more about Miraculous.

How it all works

Get an episode

To get a recommendation you need to fill (just start typing and all possible options will be shown) two fields: a similar episode and a participating character. Both of them are optional and can be left blank. Here is how the recommendation depends on the filling of this two fields:

1. If both of them are empty you will get a random episode.

2. If only a character is present you will get a random episode with that character.

3. If only an episode is present you will get the one most similar to it. It is always the same recommendation for the same source episode. (Every other episode is ranked based on its similarity to this one, so you will just get the top 1 everytime.)

4. If both are present you will get a combo: most similar to the source episode with the desired character. And again you will always get the same recommendation for the same fillings.

NB: if some character is only present in one episode, and you fill in both this character and this episode, your recommendation will be this exact episode.

Your recommendation will be in the form: picture, title, synopsis. Click on 'MORE INFO' will lead you to the fan wiki page about this episode.

Statistics

You can get four types of statistics: for a character, for an episode, for a season, and overall statistics. Just choose a desirable category.

1. For the first three categories you will get a field. You need to choose exact option for which you want to get statistics. If the field is empty you will get statistics for a random option.

2. Overall statistics you will get just by clicking on the respective category.

NB: on fan wiki some characters can be written differently on different pages (for example, Jagged or Jagged Stone or Stone), so they considered to be different characters in every place on this website.
